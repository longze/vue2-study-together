# Vue 服务器端渲染

## 首先是升级依赖

    vue & vue-server-renderer >= 2.3.0
    vue-router >= 2.5.0
    vue-loader >= 12.0.0 & vue-style-loader >= 3.0.0

如果你的项目不打算升级，那么就没有必要往下看了。
